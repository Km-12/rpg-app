# 宝探しゲーム

## 概要
https://km-12.github.io/rpg-app/  
「宝探しゲーム」は、Pythonのゲームライブラリ**Pyxel**を使って作成したゲームです。プレイヤーは宝箱を見つけるとゲームクリアです。

## 操作方法

### 1. タイトル画面
- 「**はじめる**」でゲームを開始します。

### 2. ゲーム画面
- 十字キーでキャラクターを移動します。
- 「**宝箱**」のアイテムを取ると**ゲームクリア**となります。
- 「**うごめく敵**」に当たる、「**偽宝箱**」、「**偽鍵**」を取ると**ゲームオーバー**になります。

### 3. ゲームクリア、ゲームオーバー画面
- 「**もういちど**」でスタート画面に戻ります。
